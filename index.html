Stage 1:
Mozilla + Web Literacy
Mozillians in your classroom + Mozilla learning events + Mozilla learning tools + Mozilla teaching material ≈ Mozilla Webmakers

Learning by doing!

https://thimble.webmaker.org/en-US/projects/map/edit

Mozilla Thimble is a very simple and interactive way to learn basic HTML. An editor on the right hand side of the screen lets you write things and the results are shown immediately on the left. You can also choose from already existing projects and write in them. The editor shows you when you make an error and links it to the documentation on the developer network. You can choose from various challenges to become better and better in editing HTML.

Lights, Camera, interaction!

http://mozillapopcorn.org

Mozilla Popcorn is a simple way to create interactive web experiences from video and audio materials. You have an editor to overlay web content over videos and interact with parts of the page.

X-Ray Goggles

https://webmaker.org/en-US/tools/#x-ray-goggles

Mozilla X-Ray Goggles is a tool to examine and remix pages.

Organizing events

Mentored bugs: Algorithm

			function contributor(myFavoriteTech) {
			  if (!myFavoriteTech)
			    return webmaker();
			  var bug = mentoredBugs.pick(myFavoriteTech); // FIXME: prof filter
			  var guidance = contact(bug.mentor, mozilla.irc);
			  do {
			    // work on patch
			    if (!guidance.communicate())
		          throw new ProjectFailed();

			  return bug.myCodeInFirefox;  // bragging rights!
			}

One mentor + One open-ended task + One group of students + One professor = One student project

Student Projects: Algorithm

			function students(subject, prof) {
			  var project = projects.pick(subject, prof);// or prof filters first  
			  var guidance = contact(project.mentor, mozilla.irc, prof);
			  do {
			    project.refine();
			    do {
			      // work on project
			      if (!guidance.communicate())
		            throw new ProjectFailed();
			    } while (!project.isWorking);
			  } while (!project.isComplete);

			  return project.release(); // worldwide exposure!
			}

Independent work + World-class ambitions + Passing reviews + Interacting with a team + Worldwide exposure = A great experience!
